~~~~~~~~~~~~~~~~
TLDR Quick Usage
~~~~~~~~~~~~~~~~

Export from CC3
===============

Create your character in CC3 export it using the 'Unity' option.

Import into Unity
=================

Drag all the exported files into your Unity (HDRP 2020.3+) project.


Process the Character
=====================

Open the tool window via right clicking on the fbx file or use the top menu (CC3 -> Import Character)

In the tool window click on the character you want to process (make sure it's name is the one shown at the top).  Then click 'High Quality' and wait while Unity processes the character and sets it up.  If it looks wierd, wait for any shaders to compile (progress bar at bottom right).


Thats it
========

You're all done.  Drag the chracter into your scene and do your thing.
